# What is Data Science?
From my opinion and to put it simply, Data Science is a study that involves in Maths, Programming, [Artifical intelligence (AI)](https://www.ibm.com/topics/artificial-intelligence) and Machine Learning (ML), to find hidden meaningfull from a raw data. Later the data can be used for decision making and strategic planning (e.g, from a csv file, from an excel file, etc).
