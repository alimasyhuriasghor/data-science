# What is Data Science?
From my opinion, Data Science is a study that involves in Maths, Programming, [Artificial intelligence](https://www.ibm.com/topics/artificial-intelligence) (AI) and Machine Learning (ML), to find hidden meaningfull data from a raw data (e.g, from a csv file, from an excel file, etc). Later the data can be used for decision making and strategic planning. For more reference of what Data Science is, you can visit either of these websites:
- [Wikipedia](https://en.wikipedia.org/wiki/Data_science).
- [IBM](https://www.ibm.com/topics/data-science).

# What are Artificial Intelligence and Machine Learning?
- Artificial Intelligence: Artificial Intelligence (AI) is a technology that enables computers to simulate human learning, comprehension, problem solving, decision making, creativity and autonomy. In essence Artificial Intelligence is a technology to mimick human's intelligence.
- Machine learning: According to Arthur Samuel, Machine learning is a study that gives computers the ability to learn without being explicitly programmed.
